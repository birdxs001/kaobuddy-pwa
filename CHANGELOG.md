# 更新记录

## v0.2.0 - 2026-05-31

第二版开始往“像一个真实学习工作台”的方向走：项目在侧边栏里切换，项目内按总览、资料、计划、复盘来用。

### 新增

- 默认前端统一为 React/Vite PWA，FastAPI 直接托管构建后的页面。
- 新增 Windows 双击启动脚本 `open-kaobuddy.bat`。
- 新增项目侧边栏，创建后可以直接切换不同科目。
- 新增项目内分页：总览、资料、计划、复盘。
- 新增知识模块计划看板，AI 计划可以拆成模块，模块能在待学、学习中、已完成之间拖拽。
- 新增批量资料上传，PDF 课件会读取文字层，并可用 AI 识别页面里的图表、截图、公式和扫描内容。
- 新增资料删除功能，误传的课件可以从资料库移除。
- 新增错题复盘、薄弱项记录和模考记录。
- JSON 备份升级到 v2，并兼容导入 v1 备份。

### 优化

- API 设置默认只显示平台和 API Key，Base URL / Model 收进高级设置。
- DeepSeek API Key 获取说明放在设置区下面，语气更像给同学看的提示。
- 生成计划、讲解、练习、模考前增加必填校验和更清楚的状态提示。
- 生成计划后先进入结果确认页，再由用户决定是否拆成知识模块。
- AI 输出在页面上会去掉 Markdown 符号，尽量按普通说明文字展示。
- 首页标题保留“考搭子”，删除临时营销式标题。
- 页面增加轻量动效、模块完成反馈和 AI 加载状态。

### 已知限制

- Windows 已提供双击启动脚本，但还需要在真实 Windows 机器上做一次最终可见验收。
- 模块拖拽在桌面端体验最好，手机端可通过完成按钮和状态列辅助调整。

## v0.1.0 - 2026-05-30

第一版发布，先把手机备考的最小闭环跑通。

### 新增

- FastAPI 后端，提供 AI 转发、计划生成、讲解、练习、模考、手写识别和视频导入接口。
- FastAPI 直接托管可用页面，双击 `open-kaobuddy.command` 后就能打开本地版本。
- React + Vite PWA 前端，支持手机浏览器使用和添加到桌面。
- IndexedDB 本地存储，用来保存考试项目、资料和 AI 输出。
- 自定义 OpenAI-compatible API 配置，内置 DeepSeek、Kimi、OpenAI 预设。
- 资料导入：文本、Markdown、TXT、PDF、手写图片/PDF、B站等公开视频链接。
- JSON 导出导入，方便手动备份。

### 已知限制

- 视频字幕抓取依赖公开字幕和页面可访问性，抓不到时需要手动补充资料。
- 手写 PDF 的扫描图识别还没有做逐页转图增强。
- 第一版不包含账号、云同步和多人协作。
