# Roadmap

## v0.1.0

目标：让一个人可以在手机上完成考前冲刺的最小闭环。

- 创建考试项目。
- 接入自己的 API。
- 导入资料、手写笔记和视频链接。
- 生成计划、讲解、练习和短模考。
- 本地保存，支持导出导入。

## v0.2.0

目标：把单页工具改成项目侧边栏 + 流程分页，让计划按知识点模块滚动推进，并补上 Windows 双击启动。

- Mac / Windows 都能双击启动本地服务。
- 默认入口统一到 React PWA，由 FastAPI 托管构建后的页面。
- API 设置默认只露平台和 API Key，Base URL / Model 放进高级设置。
- 首页保留“考搭子”标题，负责介绍、连接 AI 和创建第一个项目。
- 创建后通过左侧边栏切换不同科目项目。
- 项目内拆成总览、资料、计划、复盘分页。
- AI 计划可以拆成知识点模块。
- 模块支持拖拽调整顺序和状态。
- 增加错题、薄弱项和模考记录。
- 缺必填信息时给明确提示。

## v0.3.0

目标：让资料导入更稳，复盘更自动。

- 强化扫描 PDF 的手写识别。
- 给视频导入增加手动字幕编辑和来源标记。
- 给资料库增加搜索和标签。
- 薄弱项自动沉淀。
- 模考记录对比和趋势。
