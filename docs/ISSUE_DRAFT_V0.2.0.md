# [Feature] v0.2.0 项目侧边栏和知识模块计划

## 背景

第一版已经能创建项目、导入资料、接自己的 API 并生成计划。第二版要把单页工具改成更清楚的学习工作台：先连接 AI 和创建项目，再从侧边栏进入不同科目，每个科目里分总览、资料、计划和复盘。

## 目标

- [x] Mac / Windows 都能双击启动。
- [x] API 设置默认只需要平台和 API Key。
- [x] 首页保留“考搭子”标题。
- [x] 项目创建后进入左侧边栏。
- [x] 项目内有总览、资料、计划、复盘分页。
- [x] AI 计划可以拆成可拖动知识模块。
- [x] 模块能在待学、学习中、已完成之间拖拽。
- [x] 错题、薄弱项和模考记录能保存到本地。

## 验收标准

- [x] 手机端可用。
- [x] 不破坏本地存储，v1 JSON 可以导入。
- [x] API Key 不进入日志或导出文件。
- [x] 必填信息缺失时，AI 操作前有明确提示。
- [ ] 在真实 Windows 机器上双击 `open-kaobuddy.bat` 验收。
